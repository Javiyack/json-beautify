# json-beautify

MVP: Validador y formateador JSON en el navegador.

## Scripts
- dev: entorno desarrollo
- build: compilación producción
- test / test:watch: pruebas unitarias

## Uso rápido
Instala dependencias y arranca dev server.

```bash
npm install
npm run dev
```

Abre http://localhost:5173

## Estructura
- src/core: parse y utilidades
- src/main.ts: bootstrap simple
